Website created by Denis Papara as a course assignment.

This website was built in VS Code using HTML, CSS, and JavaScript.

Initially, I used HTML to create the page's "skeleton" to form a clearer idea about the project.

Later, I used CSS to modify the attributes of each element in HTML.

With the help of JavaScript and CSS, I implemented scroll animations.

To create a page with a more unique appearance, I also used animated SVGs, and some of the designs were made manually.

The goal was to create a page as unique and attractive to the human eye as possible.

History

HTML, ose HyperText Markup Language, shërben si blloku themelor i World Wide Web, duke ofruar një strukturë të standardizuar për krijimin dhe organizimin e përmbajtjes në faqet e internetit.

Zhvilluar nga Tim Berners-Lee në vitin 1991, HTML përdor një sistem etiketash për të përcaktuar dhe strukturuar elementët brenda një dokumenti.

Këto etiketa, të mbyllura në këndin e këndshëm, shënojnë komponentë të ndryshëm si tituj, paragrafe, lidhje, imazhe dhe shumë të tjera.

HTML mundëson integrimin e paqëndrueshëm të elementëve multimedialë dhe siguron kompatibilitet në të gjitha shfletuesit e internetit.

Si një pjesë kyçe e trios së bashku me CSS (Cascading Style Sheets) dhe JavaScript, HTML luajnë një rol të rëndësishëm në krijimin e faqeve të internetit dinamike dhe tërheqëse vizualisht, duke lejuar zhvilluesit të krijonjë përvoja online tërheqëse dhe interactive për përdoruesit në nivel botëror.


HTML, or HyperText Markup Language, serves as the fundamental building block of the World Wide Web, providing a standardized structure for creating and organizing content on web pages.

Developed by Tim Berners-Lee in 1991, HTML employs a system of tags to define and structure elements within a document.

These tags, enclosed in angle brackets, delineate various components such as headings, paragraphs, links, images, and more.

HTML facilitates the seamless integration of multimedia elements and ensures compatibility across different web browsers.

As a key component of the trio alongside CSS (Cascading Style Sheets) and JavaScript, HTML plays a pivotal role in the creation of dynamic and visually appealing web pages, allowing developers to craft engaging and interactive online experiences for users worldwide.
